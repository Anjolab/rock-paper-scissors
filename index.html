Rules

Rock beats Scissors - by smashing them up

Scissors beats Paper - by chopping them up

Paper beats Rock - by somehow covering it or something...

How to play

  1. Find a buddy
  2. Choose which one is bud #1 and which one is bud #2
  3. Go here: https://github.com/njall/rock-paper-scissors
  4. Bud #1: Click Fork
  5. Bud #1: Add bud #2 as a collaborator
  6. Both open up your terminal!
  7. Run this command in your terminal: git clone git@github.com:njall/rock-paper-scissors.git (where njall is bud #1 username. This downloads the code)
  8. Open and edit your weapon of choice - _data/rock.yml _data/paper.yml or _data/scissors.yml
  9. Change false to true and add your name
  10. Run this command: git add _data/file.yml (where file is your choice (e.g. rock). This 'stages' the file which tells git 'Hey, I think this file is ready to be saved') read more)
  11. Run this command: git commit -m "Added my weopon of choice" (which tells git 'yep, go ahead and save this on my machine, and here's a handy reminder of what I've done')
  12. Run this command: git push (which tells git 'now push those changes onto github for the whole world to see')
  13. ALERT: An issue occured for the slow-poke! You've both editted the same repository at the same time. You can add one persons set of changes, but the second one requires a bit of action!
  14. Slow-poke, run this command: git pull (you've pulled speedy gonzalez' changes into your repository)
  15. Now you need to accept these changes. Try commiting them again like you did when you chose your weapon the first time.
  16. All good? Push!
  17. Not good? Might be a merge error!
  18. Finally, go to njall.github.io/rock-paper-scissors to see who won!
