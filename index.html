Rules

Rock beats Scissors - by smashing them up

Scissors beats Paper - by chopping them up

Paper beats Rock - by somehow covering it or something...

Instructions

To play, pop a 'chosen: true' next to your choice of rock or paper or scissors and add your player name in the files linked below.

Edit _data/rock.yml.

Edit _data/paper.yml.

Edit _data/scissors.yml.

  1. Go here: https://github.com/njall/rock-paper-scissors
  2. Click Fork
  3. Find a bud
  4. git clone git@github.com:njall/rock-paper-scissors.git (where njall is your buds username)
  5. open and edit your choice of _data/rock.yml _data/paper.yml or _data/scissors.yml
  6. git add _data/file.yml (this 'stages' a file which tells git 'Hey, I think this file is ready to be saved') read more)
  7. git commit -m "I added my choice" (which tells git 'yep, go ahead and save this on my machine')
  8. git push (which tells git 'now push those changes onto github for everyone to see')
  9. and finally, go to njall.github.io/rock-paper-scissors
Open up your terminal
